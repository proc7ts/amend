  • Defined in base/amend-request.ts:15

Amendment request.

Contains updates to apply by resulting amendment. Also serves as modification or extension of amendment target.

The properties present here are added to new amendment target potentially replacing the original ones. It also may override a core API of the modified amendment target.

Type parameters

  • TAmended

    A type of amended entity to modify.

  • TExt = EmptyExtension

    A type of amended entity extension.

Amendment

Amendment<TAmended>: Amender<TAmended> | AmendmentSpec<TAmended>
  • Defined in base/amendment.ts:14

An amendment of some entity.

Applies to amended entities, such as classes or their members.

May be represented either by amendment action, or by its specifier. A amenderOf function can be used to convert any amendment to amendment action.

Type parameters

  • TAmended

    Amended entity type.
